# 네이버 데이터랩 개요

네이버 데이터랩은 네이버 통합검색에서 발생하는 검색어 트렌드와 쇼핑 영역의 클릭 데이터를 조회할 수 있는 서비스입니다.

## 1. 통합검색어 트렌드
통합검색어 트렌드는 네이버 통합검색에서 발생하는 검색어를 연령별, 성별, 기기별(PC, 모바일)로 세분화해서 조회할 수 있는 API입니다.

### 주요 기능
- **주제군 설정**: 분석하고 싶은 주제어와 하위 주제어들을 그룹화하여 합산 검색 추이를 조회할 수 있습니다.
- **세분화된 정보**: 기간(일간, 주간, 월간), 연령(5살 단위), 성별, 기기별 정보를 제공합니다.
- **상대적 값 제공**: 요청 기간 중 검색 횟수가 가장 높은 시점을 100으로 둔 상대적 지표를 제공합니다.

## 2. 쇼핑인사이트
쇼핑인사이트는 네이버쇼핑 서비스의 검색 결과에서 사용자가 클릭한 데이터를 분야별, 연령별, 성별, 기기별로 세분화하여 조회할 수 있는 API입니다.

### 주요 기능
- **분야별 통계**: 네이버쇼핑의 4단계 카테고리 구조를 기반으로 세부 통계 정보를 제공합니다.
- **검색어 클릭 추이**: 특정 분야나 상품군에 대한 사용자 클릭 트렌드를 파악하여 재고 관리나 마케팅에 활용할 수 있습니다.
- **주요 구매층 파악**: 타깃 연령층과 성별의 비중 및 추이를 분석할 수 있습니다.
