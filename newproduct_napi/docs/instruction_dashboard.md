# 작업지시서: 우유거품기 트렌드 분석 Streamlit 대시보드 구축

본 문서는 수집된 네이버 API 데이터(쇼핑 트랜드, 블로그 검색, 쇼핑 상품 검색)를 시각화하고 분석하기 위한 Streamlit 대시보드 개발 가이드라인입니다.

## 1. 프로젝트 개요
- **목적**: 우유거품기 시장 트렌드 파악 및 경쟁 제품 분석을 위한 인터랙티브 대시보드 구축
- **주요 도구**:
  - **Framework**: Streamlit
  - **Visualization**: Plotly (Express & Graph Objects)
  - **Data Handling**: Pandas
  - **Environment**: `uv`를 활용한 가상환경 관리

## 2. 대시보드 구조 및 메뉴 구성

대시보드는 좌측 사이드바와 메인 화면(탭 구조)으로 구성합니다.

### 사이드바 (Sidebar)
- **키워드 필터**: '우유거품기', '밀크포머', '밀크프로더', '에어로치노', '우유스팀기' 중 분석 대상 선택 (다중 선택 가능)
- **데이터 로드 버튼**: 최신 수집된 CSV 파일을 자동으로 병합하여 로드

### 메인 화면 (Tabs)
1. **[Tab 1] 검색 트랜드 비교**: 키워드별 검색량 변화 시각화
2. **[Tab 2] 블로그 여론 분석 (EDA)**: 블로그 검색 결과 기반 텍스트 분석 및 정보성 데이터 확인
3. **[Tab 3] 쇼핑 시장 분석 (EDA)**: 상품 가격대, 브랜드 점유율 등 경쟁 환경 분석

## 3. 필수 구현 리스트 (그래프 및 표)

### 시각화 (Plotly Graph) - 5가지 이상
1. **[라인 차트]** 일별/월별 검색 트랜드 변화 (키워드별 비교)
2. **[바 차트]** 수집된 상품들의 브랜드별 점유율 (Top 10)
3. **[히스토그램]** 상품 가격(lprice) 분포도 및 이상치 확인
4. **[파이 차트]** 네이버 쇼핑 수집 상품의 카테고리/몰별 비중
5. **[산점도]** 브랜드별 가격대와 리뷰수(있을 경우) 또는 상품 타입 간의 관계

### 데이터 요약 (Tables) - 5가지 이상
1. **[상세 리스트]** 선택된 키워드의 전체 데이터 원본 (필터링 기능 포함)
2. **[기술통계 표]** 가격(lprice)에 대한 평균, 중위수, 최저/최고가 요약
3. **[빈도 요약]** 블로그 제목 내 주요 키워드 추출 빈도 (Top 30)
4. **[랭크 표]** 최저가 기준 가성비 상위 10개 상품 리스트
5. **[비교 표]** 키워드별 평균 가격 및 상품 수 비교 요약표

## 4. 개발 지침 (Checklist)
- [ ] **한글 폰트 설정**: Plotly 차트 내 한글 깨짐 방지 설정 적용
- [ ] **데이터 병합**: `data/processed` 내 여러 CSV 파일을 키워드 기준으로 스마트하게 병합하여 사용
- [ ] **인터랙션**: 사용자가 사이드바에서 키워드를 변경할 때마다 모든 그래프와 표가 실시간으로 업데이트되도록 구성
- [ ] **에러 핸들링**: 데이터가 없는 경우(예: 쇼핑 트랜드) "데이터가 존재하지 않습니다"라는 안내 문구 출력
- [ ] **디자인**: `st.columns`를 활용하여 지표(Metric) 카드 등을 배치, 가독성 높은 레이아웃 구현

## 5. 실행 방법
```bash
python -m uv run streamlit run app.py
```
